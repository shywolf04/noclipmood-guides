    Tools

    • Freecam by IDK31
      Features: Free Camera, FOV, Slowmotion, HUD toggle, tonnemapping, DOF, and more effects.
    • No Breathing Sounds Mod by ShyWolf

    Usage

    Once you load the cheat engine table you will need to click the boxes of the loaded table the features you want to use for using them.

    Example: If you need to move and disable crosshair, then tick CAMERA CONTROLS, MISC >> Main Menu Option >> Double click value then set to none

    Custom in-game bindings

    To make things easier, you can set custom bindings by modifying the TdInput.ini file, located in C:\Users\{currentuser}\Documents\EA Games\Mirror's Edge\TdGame\Config.

    New commands need to be added under bEnableMouseSmoothing=true

    Example:

    [Engine.PlayerInput]
MoveForwardSpeed=1200.000000
MoveStrafeSpeed=1200.000000
LookRightScale=12000.000000
LookUpScale=-12000.000000
MouseSensitivity=27.999998
DoubleClickTime=0.250000
bEnableMouseSmoothing=true
Bindings=(Name="H",Command="killbots",Control=False,Shift=False,Alt=False)
Bindings=(Name="J",Command="suicide",Control=False,Shift=False,Alt=False)

    Essential in-game bindings

    Kill bots: Bindings=(Name="H",Command="killbots",Control=False,Shift=False,Alt=False)

    Suicide: Bindings=(Name="J",Command="suicide",Control=False,Shift=False,Alt=False)

    If you want to look at other commands, check out this list made by LucasOe on GitHub Gist.

    Tips

    If you're stuck after switching freecam, try to kill all bots and switch again or die instantly.
